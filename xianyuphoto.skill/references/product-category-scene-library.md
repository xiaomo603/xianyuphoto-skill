# 不同产品类目的场景词库

匹配一个主要类目，并按顺序将三组推荐组合分别用于三张主图。

## A. 厨房 / 家用厨具类

适合：锅具、电蒸锅、电煮锅、料理机、碗架、置物架、厨房收纳等。

1. 厨房台面 + 自然光 + 轻微生活感
2. 出租屋小厨房 + 干净台面 + 手机实拍
3. 厨房角落 + 靠墙摆放 + 产品完整展示

## B. 商用餐饮类

适合：商用电蒸锅、大锅、蒸饭柜、餐饮设备、食品加工设备、货架等。

1. 早餐店后厨 + 不锈钢操作台 + 真实使用痕迹
2. 包子铺蒸台旁 + 商用环境 + 手机实拍
3. 店铺仓库角落 + 产品完整摆放 + 干净不杂乱

## C. 家居生活类

适合：收纳用品、桌椅、小家具、清洁用品、生活小物、家用设备等。

1. 客厅角落 + 自然光 + 干净背景
2. 阳台角落 + 真实生活感 + 产品完整展示
3. 出租屋卧室 + 地面摆放 + 手机随手拍感

## D. 收纳 / 置物架 / 家具类

适合：货架、置物架、收纳柜、鞋架、折叠桌、椅子、家用架子等。

1. 墙边摆放 + 产品占画面 70% + 空间参照物
2. 阳台收纳区 + 有少量生活用品 + 不杂乱
3. 仓库地面 + 真实大件感 + 手机实拍

## E. 五金工具 / 设备类

适合：工具箱、电钻、维修工具、五金配件、工地设备、小型机器等。

1. 维修台面 + 少量工具散落 + 产品清晰居中
2. 车库工作台 + 自然光 + 真实使用感
3. 水泥地面 + 产品完整展示 + 不脏乱

## F. 养殖 / 农用类

适合：羊槽、鸡笼、饲料槽、养殖设备、农具、围栏用品等。

1. 羊圈旁边 + 两只羊自然吃饲料 + 产品清晰
2. 农家院水泥地 + 饲料袋旁 + 真实手机拍摄
3. 养殖场围栏边 + 自然光 + 不杂乱背景

## G. 办公 / 数码 / 桌面类

适合：电脑配件、打印机、办公用品、小设备、数码支架、显示器周边等。

1. 办公桌面 + 电脑旁 + 产品清晰
2. 家用书桌 + 自然光 + 轻微桌面杂物
3. 工位桌面 + 手机实拍 + 不要广告感

## H. 大件 / 重货 / 商用设备类

适合：大锅、大型货架、机器设备、商用柜、大型桶、大型工具等。

1. 仓库地面 + 纸箱参照 + 产品显大
2. 店铺角落 + 真实商用环境 + 不杂乱
3. 厂房门口 + 自然光 + 产品完整展示
