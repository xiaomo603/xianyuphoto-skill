# 闲鱼主图提示词模板

为每张图片单独组装完整提示词。不要把三个场景合并进同一个生成任务。

## 变量

- `{比例}`：默认 `3:4`，用户指定时为 `1:1`。
- `{产品名称}`：用户提供的名称，或根据图片推断的不含品牌的通用名称。
- `{目标买家}`：用户提供或根据图片与核心卖点推断。
- `{核心卖点}`：用户提供的 1–3 个卖点。
- `{场景词}`：匹配类目中对应序号的推荐组合。
- `{展示目的}`：该张图片需要突出的使用价值或细节。

## 单张图片模板

```text
请基于我上传的产品主体实拍图，生成 1 张适合闲鱼商品主图使用的真实图片，比例为 {比例}。

产品名称：{产品名称}
目标买家：{目标买家}
核心卖点：{核心卖点}
使用场景：{场景词}
展示目的：{展示目的}

Use the object in the reference image(s) as the same identity. Do not redesign it or create a similar-looking new object.

参考图中的产品是唯一主体，必须保持产品本身高度一致。只允许更换背景、环境、拍摄场景、摆放位置、拍摄角度和光线。不要重新设计产品，不要替换成相似款，不要改变产品的外观、结构、颜色、材质、比例、尺寸关系、边缘轮廓、表面纹理、Logo、按钮、接口、开口、配件和其他细节。

图片要像普通卖家用手机实拍，不要像广告图、电商精修图、3D 渲染图或 AI 生成图。产品必须清晰完整，不能变形、模糊、漂浮、裁切、重复生成或被遮挡。产品和背景要自然融合，具有真实光影、接触阴影和正确透视。

不要添加任何文字、水印、贴纸、价格、促销标签、箭头、夸张光效、无关人物或无关杂物。
```

## 三张图的展示目的

1. 突出产品在匹配场景中的日常实用性。
2. 突出产品在匹配场景中的耐用、方便或省心。
3. 突出产品的材质、结构、容量、做工或功能区域，同时保持整个产品完整可见，不得裁切成局部特写。
